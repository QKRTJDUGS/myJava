======================================================
  SQL*Plus SPOOL 사용법 - 공부 내용 자동 저장 가이드
======================================================

[사용법]
SQL*Plus를 열고 아래 순서대로 입력하세요.

1. 공부 시작 전 (파일 기록 시작):
   SQL> SPOOL C:\JavaWorks\myJava\sql_logs\study_20260330.sql

   * 날짜는 오늘 날짜로 변경하세요 (YYYYMMDD 형식)

2. 평소대로 공부 진행:
   SQL> SELECT * FROM dual;
   SQL> DESC emp;
   SQL> SELECT empno, ename, sal FROM emp WHERE sal > 2000;

3. 공부 끝날 때 (파일 기록 종료):
   SQL> SPOOL OFF

이렇게 하면 study_20260330.sql 파일이 자동으로 저장되고,
auto_push.py가 감지해서 GitHub에 자동으로 올려줍니다!

------------------------------------------------------
[파일 이름 예시]
  study_20260330.sql   ← SQL 공부 내용
  study_20260331.txt   ← .txt 확장자도 OK
------------------------------------------------------

[주의사항]
- SPOOL 경로에 한글이 포함되면 오류가 날 수 있습니다
- 파일 이름에 날짜를 넣으면 날짜별로 정리됩니다
- SPOOL OFF 를 꼭 실행해야 파일이 완전히 저장됩니다
